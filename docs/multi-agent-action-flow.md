# Multi-Agent Action Flow

Non-operative sandbox artifact.

```text
Agent A (Planner)
Agent B (Researcher)
Agent C (Coder)
Agent D (Reviewer)
Agent E (Ops)
        ↓
Proposed Action
        ↓
ContinuityOS
        ↓
VALID | NULL
        ↓
Execution
        ↓
Proof
        ↓
Record & Reconcile
```

Core separation:
- Proposal ≠ authority
- Capability ≠ permission
- Validation ≠ execution
- Visibility ≠ legitimacy

Invariant:
If no valid object exists → nothing happens.

Purpose:
Educational sandbox representation of a multi-agent governed action path.
